## 一、核心目标
在末日环境中通过管理状态、收集物资、应对事件，尽可能延长生存时间，解锁成就并达成理想结局。


## 二、基础机制
1. **时间推进**：以"天数"为单位循环，每天需选择行动，结束后状态自然衰减并可能触发随机事件。
2. **状态管理**：
   - 核心状态：健康值(0-10)、饥饿值(0-6)、口渴值(0-6)，归零可能导致死亡
   - 负面状态：生病（健康每日-0.5，饥饿/口渴消耗加剧）、受伤（健康每日-0.3）、极度饥饿/口渴（持续扣血）
   - 季节影响：每30天一个季节，冬季（第3季）会增加极寒事件概率


## 三、每日行动选择
1. **搜寻物资**：高风险高收益，随机触发正面/负面事件（如发现食物、遭遇野兽），可能获得道具或状态变化。
2. **休息恢复**：健康值+1，饥饿/口渴-0.5，30%概率恢复生病/受伤状态。
3. **使用道具**：消耗道具恢复状态或触发效果（如罐头补饥饿、急救包清负面状态）。
4. **保存游戏**：将当前进度保存为JSON文件，方便后续导入。
5. **退出游戏**：结束当前进程（未保存进度会丢失）。


## 四、关键系统
1. **事件系统**：
   - 正面事件：如"发现水源"（口渴+1）、"遇到友善幸存者"（获罐头）、"天气转好"（提升正面事件概率）。
   - 负面事件：如"野兽攻击"（健康-2+受伤）、"食物中毒"（健康-1+生病）、"恶劣天气"（提升负面事件概率）。
   - 特殊事件：如"封锁薄弱点"（可尝试突围，10%成功率达成结局）、"军方来到"（获物资并解锁军方救援可能）。

2. **道具系统**：
   - 基础道具：罐头（补饥饿）、水瓶（补口渴）、绷带（治受伤）、急救包（补健康+清负面）。
   - 功能道具：手电筒/电池（提升正面事件概率）、刀具（降低负面事件概率）、帐篷（降低生病概率）等。
   - 特殊道具：猫咪（解锁猫咪结局）、电脑（可发送求救信号解锁电子救援结局）、物资箱（开3件随机道具）。

3. **成就系统**：
   - 生存类：如"生存达人"（存活100天）、"极限生存者"（极难难度存活30天）。
   - 收集类：如"收藏家"（拥有所有道具）、"猫咪守护者"（获得猫咪）。
   - 行为类：如"幸运儿"（连续5次正面事件）、"突围者"（成功冲出封锁区）。

4. **存档与难度**： 
   - 支持手动保存/导入JSON格式存档。
   - 6种难度（新手到极其困难）影响道具概率、状态消耗、事件概率。
   - 2种角色选择：父亲（提升道具概率+饥饿耐力）、母亲（提升道具概率+降低负面事件）。


## 五、结局触发
1. **生存结局**：存活200天可选择结束或继续挑战。 
2. **猫咪结局**：拥有猫咪且存活50天以上，5%概率触发（猫咪吸引救援）。
3. **军方结局**：触发"军方来到"事件后，每日1/500概率被救援。
4. **电子救援结局**：使用电脑发送求救信号，1%概率被科考队救援。
5. **极寒结局**：冬季期间，每日1/1000概率被极地救援队发现。
6. **冲出封锁区**：遇到薄弱点时选择突围，10%成功率进入安全区。
7. **隐藏结局**：暂无发现玩家，有玩家发现即公开

## 六、失败条件
- 健康值≤0
- 饥饿值≤0且处于"极度饥饿"状态
- 口渴值≤0且处于"极度口渴"状态
